About Me

Hi, I'm Tierra!

During the day I'm an intrepid science teacher but in the evenings I've been learning to program. I started with Python because I heard that it was an easy language for beginners. After I had been learning awhile I decided to attempt a personal project using Python. I wanted to transfer my music library from Spotify to Google Play Music. Unfortunately, there was not system to do this and my library was huuge. So I decided to create one using Python! This was a challenging project for me and I learned a ton! Even better, it worked!

Lately my focus has shifted a little to web development. I started on FreeCodeCamp but I wanted to kick things up a notch by joining a real, in-person bootcamp. It’s only been a week, but so far, I’m having a ton of fun! Things are challenging (I’m looking at you CSS) but it feels really great when things work and I get to make cool things like this website! And when I go on Youtube and see all of the cool things that other people can do, I get really excited about the future. I can't wait to learn all of the things!
